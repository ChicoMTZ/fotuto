Welcome to Fotuto's documentation!
==================================

**Fotuto** is a web SCADA to monitor variables, control actions and notifications. Created by `Julio Cesar Rodriguez <mailto:juliocesarrodriguezcruz@gmail.com>`_.
It is free and the source code is located in https://github.com/cesarcruz/fotuto.
